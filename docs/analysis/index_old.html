Computing and visualizing PCs using rsvd and ggplot2: an illustration with the Iris data¶

Quick Start¶

  1. Install the required packages from CRAN.

    install.packages(c("repr","ggplot2","rsvd"))

  2. Walk through the first notebook to compute the PCs using the rsvd package.

  3. Walk through the second notebook to generate evocative visualizations of the PCA results using ggplot2.
